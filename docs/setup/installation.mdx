## ServeQuery

`ServeQuery` is available as a Python package. Install it using the **pip package manager**:

```python
pip install servequery
```

To install `servequery` using **conda installer**, run:

```sh
conda install -c conda-forge servequery
```

## ServeQuery LLM

To run evaluations specific to LLMs that include additional dependencies, run:

```python
pip install servequery[llm]
```

## Tracely

To use tracing based on OpenTelemetry, install the sister package **tracely**:

```sh
pip install tracely
```
